# Draw - Landing Page Professionnelle

## Description
Landing page moderne et professionnelle pour Draw, une agence de design et développement web. Site optimisé avec design premium en noir, blanc et bleu électrique.

## Structure
- **index.html** - Structure principale avec toutes les sections
- **style.css** - Styles CSS purs, design moderne et responsive
- **script.js** - JavaScript pour interactions et formulaire

## Sections
1. Hero - Message d'accueil avec CTA
2. Services - Présentation des 3 services principaux
3. Pourquoi Draw - 4 avantages clés
4. Portfolio - 6 projets de réalisation
5. Témoignages - 3 avis clients
6. Contact - Formulaire fonctionnel
7. Footer - Informations légales

## Configuration du formulaire
Le formulaire de contact nécessite une configuration EmailJS ou FormSubmit. Consultez les instructions détaillées dans script.js.

## Technologies
- HTML5 pur
- CSS3 pur (sans framework)
- JavaScript vanilla
- EmailJS pour l'envoi d'emails (à configurer)

## Couleurs
- Primaire: #005CFF (bleu électrique)
- Noir: #000000
- Blanc: #ffffff

## Responsive
Entièrement responsive, testé pour mobile et desktop.
